<h3 id="用户对网络安全的需求">用户对网络安全的需求</h3>
<ul>
<li><strong>可用性</strong>：在可能发生的突发事件情况下，计算机网络仍然可以处于<strong>正常运转状态</strong>，用户可以使用各种网络服务</li>

</ul>
